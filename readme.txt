== Description ==

The Bet sport Free plugin allows you to publish matches of different sports organized by leagues, shows the match date, team logo, stadium, group, etc.

Plugin Features:

You can post matches
- You can register leagues
- You can register equipment
- You can see the bets of each user
- You can show the matches with the use of a shortcode

== Setting-up ==

Download and unzip the plugin file.
Upload the plugin folder to the plugins directory of your WordPress site.
Activate the plugin via the "Plugins" menu in WordPress.
